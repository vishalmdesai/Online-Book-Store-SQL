# 📊 Online Book Store Data Analysis using SQL

## 📌 Overview
This project analyzes an online book store dataset to extract business insights using SQL.

## 🛠️ Tools Used
- PostgreSQL (PGAdmin)
- SQL

## 📊 Key Analysis
- Book sales by genre
- Customer purchase patterns
- Revenue analysis
- Inventory tracking

## 📁 Dataset
Includes:
- Books
- Customers
- Orders

## 🔍 Key Insights
- Top-selling books identified
- High-value customers detected
- Stock tracking after orders

## 🚀 Outcome
Applied SQL concepts like JOIN, GROUP BY, HAVING, and window functions.

## 🔗 Project Files
- SQL Queries → `/sql`
- Dataset → `/data`
